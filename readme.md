# RusFritta

Сайт для продажи фритт (грунтовые, покровные, керамические и специальные).  

## Структура сайта

- index.html — Главная страница  
- catalog.html — Каталог фритт  
- style.css — Стили сайта  
- script.js — Скрипты для интерактивности  
- /img — Изображения продукции  
- RusFritta_catalog.pdf — Полный PDF-каталог  
- favicon.ico — Иконка сайта
